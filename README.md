# F-Lewis-Medical-Centre
A three tier, role based web application for F Lewis Medical Centre for staff to manage personal and appointment data. Include a login page with MFA and hashed password protection. Presentation Layer: HTML/CSS/JavaScript for user interaction. Application Layer PHP controllers, models and security modules. Data Layer: MySQL/PostgreSQL for persistant storage. Security features such as Password hashing (bcrypt/Argon2) MFA support, Session Hardening (HttpOnly, secure SameSite), Parameterised queries (PDO) Output encoding (XSS mitigation) server side validation Environmental variables for secrets Logging and monitoring and generic error messaging. The system aligns with ISO/IEC 27001 Clauses 4-10 and Annex A Controls.
